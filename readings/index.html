About Reading List Projects

2023

  • The Pragmatic Programmer by Andy Hunt and Dave Thomas is a classic guide for software developers. The book provides actionable advice on a wide range of topics, including project planning, execution, testing, debugging, and automation, and has become a staple resource for programmers at all levels of experience.
  • Python Crash Course by Eric Matthes is a comprehensive introduction to Python programming for beginners. With hands-on projects and practical exercises, the book aims to teach readers the fundamentals of Python programming in a clear and concise manner. It is a great resource for anyone who wants to learn Python, whether they have prior programming experience or not.
